Suchfunktion

April 28, 2021 · 1 min · tuxifreund

Bei der Überlegung, wie ich für dieses Blog eine Suchfunktion einbauen könnte, bin ich in der tollen Doku dieses wirklich wundervollen Themes darauf gestossen, dass ein Template für die Suche fest eingebaut ist. Das habe ich so auch eingebaut und die Suche findet sich nun hier oder auch oben im Menü. Die Umsetzung wird hier mithilfe von einer JSON-Datei durchgeführt, die von einem JavaScript-Code durchsucht wird. Deshalb muss mindestens für diese eine Seite JavaScript aktiviert sein - oder man benutzt halt z.B. die javascriptfreie Variante von DuckDuckGo.
