Mein Problem mit Github

July 7, 2021 · 1 min · tuxifreund

Microsoft hat ein echtes Talent, sich bei seinen Benutzern unbeliebt zu machen. Respekt!

Das Github nicht Open-Source ist und von Microsoft gekauft wurde, war mir bewusst und ich hoste trotzdem meinen Code und auch diesen Blog auf Github. Mal sehen, wie lange noch.

In den Augen der Redmonder ist echtes Programmieren also überflüssig und man schreibt eine ganz tolle “Künstliche Intelligenz”, die nun Projekte auf Github durchsucht und den entsprechenden Code liefert. Warum sollte die KI, die Open-Source-Software durchsucht, nicht auch freie Software sein? Am Ende wird die ja auch von einer KI kopiert. Nein, das ginge ja ganz und gar nicht!

Ich schreibe also Programme, stelle sie unter die GPL und weiß nicht, ob diese “Intelligenz” nicht meinen Code irgendwo anders vorschlägt. Da könnte ich mir dann auch die GPL sparen. Ich bin aber auch kein Lizenz-Experte…

Ich gehe also davon aus, dass meine Projekte und dieser Blog in Kürze auf eine andere Plattform umziehen. Ob dies dann aber eine bereits bestehende Gitlab- oder Gitea-Instanz wird oder ob ich da nicht selbst etwas hoste, da bin ich mir aber noch nicht sicher. Der Github-Account bleibt aber auf jeden Fall, weil manche Projekte, an denen ich mitarbeite noch auf Github liegen.
